# 제품 설계

## 한 줄 정의

출근도장은 소규모 사업장이 무료로 쓸 수 있는 웹 기반 큐알 출퇴근 기록기다.

## 사용자

- 사장님: 사업장, 직원 슬롯, 키오스크, 근태표를 관리한다.
- 직원: 앱 설치 없이 등록 링크와 패스키로 자기 기기를 묶고 출퇴근을 찍는다.

## MVP 범위

1. 사업장 생성
2. 직원 슬롯 생성
3. 직원 일회용 등록 링크
4. 패스키 기반 직원 기기 바인딩
5. 현장 큐알 출퇴근
6. 위치 확인
7. 월별 근태표
8. CSV 내려받기
9. 정정 이벤트 기록

## 제외 범위

- 급여 계산
- 주휴수당 계산
- 알림톡
- 전자계약
- 노무 상담
- 복잡한 승인 흐름

## 핵심 판단

무료 서비스는 기능이 많으면 오래 못 간다. 첫 버전은 근태 기록과 조작 방지 흔적만 남긴다.

## 편의성 스윗스팟

기본 흐름은 `공용 키오스크 큐알 + 직원 개인폰 패스키 + 위치 소프트 체크`다.

- 직원은 앱 설치 없이 개인폰 카메라로 큐알을 찍고 생체 인증만 한다.
- 사장님은 전용 장비 없이 남는 태블릿, 폰, 컴퓨터를 키오스크로 쓴다.
- 큐알은 첫 스캔 시 전역 1회 소비하고, 키오스크는 즉시 새 큐알을 표시한다.
- 위치는 기본 차단보다 위험 기록으로 먼저 남기고, 사업장 설정으로 엄격 모드를 켤 수 있게 한다.
- PIN, 셀피, 비콘, NFC는 기본값에서 뺀다. 편하지만 조작에 약하거나 설치 부담과 개인정보 부담이 크다.
