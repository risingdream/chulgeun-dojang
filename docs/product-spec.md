# 제품 설계

## 한 줄 정의

출근도장은 소규모 사업장이 무료로 쓸 수 있는 웹 기반 큐알 출퇴근 기록기다.

## 사용자

- 사장님: 사업장, 직원 슬롯, 키오스크, 근태표를 관리한다.
- 직원: 앱 설치 없이 등록 링크와 패스키로 자기 기기를 묶고 출퇴근을 찍는다.

## MVP 범위

1. 사업장 생성
2. 직원 슬롯 생성
3. 직원 일회용 등록 링크
4. 패스키 기반 직원 기기 바인딩
5. 현장 큐알 출퇴근
6. 위치 확인
7. 월별 근태표
8. CSV 내려받기
9. 정정 이벤트 기록

## 제외 범위

- 급여 계산
- 주휴수당 계산
- 알림톡
- 전자계약
- 노무 상담
- 복잡한 승인 흐름

## 핵심 판단

무료 서비스는 기능이 많으면 오래 못 간다. 첫 버전은 근태 기록과 조작 방지 흔적만 남긴다.
